# Secure and Transparent Voting System

A full-stack web-based voting system built with Python Flask and vanilla JavaScript.

## Live Demo
- Voter Page: https://voting-system-web3.netlify.app/voting_page.html
- Admin Dashboard: https://voting-system-web3.netlify.app/admin/admin_dashboard.html

## Tech Stack
- Backend: Python + Flask + SQLite
- Frontend: HTML + CSS + JavaScript
- Chart: Chart.js
- Backend hosted on Render
- Frontend hosted on Netlify

## Features
- Voter registration and login
- Live vote results with animated progress bars
- Admin election control panel
- Vote history table with voter names and timestamps
- Double voting prevention
- Toast notifications
- Professional dark theme UI
